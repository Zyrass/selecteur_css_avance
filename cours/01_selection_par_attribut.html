01 sélection par attribut

I. Sélection par attribut avec valeur exacte

	[attribut="value"]		<- Explication
	[type="email"]			<- Exemple

II. Sélection par attribut avec début de la valeur

	[attribut^="value"]		<- Explication
	[href^="http"]			<- Exemple

III. Sélection par attribut avec la fin de la valeur

	[attribut$="value"]		<- Explication
	[href^=".com"]			<- Exemple

Pratique pour les images et définir une petite puce 'png' ou 'jpg'

IV. Sélection par attribut avec une partie de la valeur

	[attribut*="value"]		<- Explication
	[href*="www.facebook.com"]	<- Exemple

02 sélection par référence

I. Sélection d'enfant direct

	element > element		<- Explication
	.container > *			<- Exemple

II. Sélection de voisin direct

	element + element		<- Explication
	h1 + h2				<- Exemple

III. Sélection de voisin directe ou indirect

	element ~ element		<- Explication
	h1 ~ h2				<- Exemple

Pratique pour les images et définir une petite puce 'png' ou 'jpg'

03 sélection par pseudo-classes

I. :first-child et :last-child

	element:first-child		<- Explication
	li:first-child			<- Exemple		

	element:last-child		<- Explication
	li:last-child			<- Exemple

II. :only-child

	element:only-child		<- Explication
	li:only-child			<- Exemple

III. :empty

	element:empty		<- Explication
	li:empty		<- Exemple

Pratique pour les images et définir une petite puce 'png' ou 'jpg'

IV. :nth-child()

	element:nth-child( 3 arguments possible )		<- Explication
	li:nth-child(even)	<- Exemple	even = paire
	li:nth-child(odd)	<- Exemple	even = impaire

	element:nth-child(an+b)	<- Explication
	li:nth-child(3n)	<- Exemple ici 3*3
	li:nth-child(3n+1)	<- Exemple ici 3*3 + 1

V. :nth-of-type()

Fait exactement la même chose que 'nth-child()' à la différence que si un quelconque element est ajouter et casse ainsi la structure, 'nth-of-type' permettra de garder toujours la structure définit sur l'élement.

	element:nth-of-type( 3 arguments possible)		<- Explication
	li:nth-of-type	<- Exemple

VI. :not()

	element:not( element )		<- Explication
	ul:not(li)			<- Exemple

04 sélection par pseudo-éléments

I. ::first-line

	element::first-line		<- Explication
	p::first-line			<- Exemple

II. ::first-letter

	element::first-letter		<- Explication
	p::first-letter			<- Exemple

III. ::before et ::after

	element::before		<- Explication
	h1:before {
		content: '1. ';			<--- ajoute un petit 1. suivit d'un espace
		content: '';			<--- utiliser pour définir son propre style
		contet: url(facebook.png);	<--- Ajoute une image png
		content: attr(attribut html); 	<--- Affiche un attribut html
	}
	
	exemple :
	
		[href*="facebook.com"]::before {
			content: url(facebook.png);
	}
	

	element::after		<- Explication
	C'est exactement la même chose que before mais tout s'affichera après
